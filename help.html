1. 개요

Seller Dashboard Pro는 셀러와 상품을 등록하고, 행사일/매출 기록을 누적하여 기간별 매출 집계, 날짜별 일정 확인, 상품/셀러 분석, 엑셀 백업 및 캘린더 출력 기능을 제공합니다.

본 시스템은 입력한 데이터를 브라우저 로컬스토리지(LocalStorage)에 저장합니다. 정기적으로 엑셀 저장을 수행하여 백업하십시오.

2. 빠른 시작

  1. 신규 추가를 클릭하여 셀러/상품/기록을 등록하십시오.
  2. 상단 연도/월로 조회 기간을 설정하십시오.
  3. 셀러 또는 상품 뷰에서 현황을 확인하십시오.
  4. 캘린더에서 날짜별 기록을 확인하고 필요 시 드래그로 날짜를 변경하십시오.
  5. 요약에서 총매출 및 랭킹을 확인하십시오.
  6. 저장으로 엑셀 백업을 수행하십시오.

3. 화면 구성

3.1 상단 헤더

영역설명
기간 필터 연도/월 선택으로 기간 집계를 설정합니다. 전체는 필터를 초기화합니다. ◀/▶로 월 이동을 수행합니다.
뷰 전환 👤 셀러 / 📦 상품 / 📅 캘린더 / 📊 요약 화면을 전환합니다.
셀러 필터 셀러 뷰에서 전체 / 상품 있음 / 상품 없음 필터링을 수행합니다.
검색 셀러/상품 뷰에서 셀러명/상품명 기준 검색을 수행합니다.
데이터 관리 저장(엑셀 내보내기), 불러오기(엑셀/CSV 가져오기), 캘린더 프린트(A4 출력), 초기화(전체 삭제)를 수행합니다.

4. 기간 필터 사용

  • 연도+월 선택 시 해당 월 기준으로 집계합니다.
  • 연도만 선택 시 해당 연도 기준으로 집계합니다.
  • 연/월 미선택 시 전체 기간 기준으로 집계합니다.
  • 전체 버튼을 클릭하면 필터가 초기화됩니다.
  • ◀/▶ 버튼으로 월 이동을 수행합니다.
기간 필터는 셀러/상품 카드, 상세 모달 집계, 요약 리포트, 캘린더 표시 모두에 동일하게 적용됩니다.

5. 데이터 등록

5.1 신규 추가

  1. 상단 신규 추가를 클릭하십시오.
  2. 셀러명(필수)을 입력하십시오.
  3. 필요 시 상품명/행사일/매출액을 입력하십시오.
  4. 추가하기를 클릭하여 등록하십시오.

5.2 입력 규칙

  • 셀러명만 입력하면 셀러만 등록됩니다.
  • 셀러명+상품명을 입력하면 셀러-상품 연결이 생성됩니다.
  • 셀러명+상품명+(행사일 또는 매출액)을 입력하면 일정/매출 기록(로그)이 생성됩니다.
매출액 단위는 억원입니다. 예: 1.25 입력 시 1.25억원으로 저장됩니다.

6. 셀러 뷰(👤 셀러)

6.1 셀러 카드

  • 카드는 기간 기준 상품 목록(최대 5개 표시)과 총 매출을 표시합니다.
  • 카드를 클릭하면 셀러 상세 모달이 열립니다.

6.2 셀러 상세 모달

셀러 상세 모달은 행사상품 탭과 일정/매출 기록 탭으로 구성됩니다.

6.3 행사상품 탭

  • 선택된 기간에 기록이 존재하는 상품을 표시합니다.
  • 상품 칩을 클릭하면 일정/매출 기록 탭으로 이동하여 관련 기록을 확인할 수 있습니다.

6.4 일정/매출 기록 탭

  1. 기록 추가 시 행사일, 상품, 매출액을 입력한 후 기록 추가/수정을 클릭하십시오.
  2. 기록 수정 시 하단 목록에서 항목을 선택한 후 값을 수정하고 기록 수정을 클릭하십시오.
  3. 선택 해제 시 동일 항목을 다시 클릭하여 입력란을 초기화하십시오.
  4. 삭제 시 항목 우측 X 버튼을 클릭하여 제거하십시오.

7. 상품 뷰(📦 상품)

  • 상품 카드는 기간 기준 셀러 목록(최대 5개 표시)과 상품 총 매출을 표시합니다.
  • 카드를 클릭하면 상품 분석 모달이 열립니다.

7.1 상품 분석 모달

  • 선택된 상품의 셀러별 매출을 내림차순으로 표시합니다.
  • 목록에서 셀러를 클릭하면 해당 셀러 모달로 이동합니다.

8. 캘린더 뷰(📅 캘린더)

8.1 일정 표시

  • 해당 월의 날짜별 일정/매출 기록을 표시합니다.
  • 이벤트는 셀러-상품-매출 형식으로 표시됩니다.
  • 일정이 많은 날짜는 +N개 버튼으로 축약 표시될 수 있습니다.

8.2 날짜별 일정 확인

  • 일정이 있는 날짜 칸을 클릭하면 하단에 해당 날짜 일정 목록 패널이 표시됩니다.
  • 일정 항목을 클릭하면 해당 셀러 모달로 이동합니다.

8.3 드래그 앤 드롭(날짜 변경)

  • 이벤트를 드래그하여 다른 날짜에 드롭하면 해당 기록의 날짜가 변경됩니다.
  • 변경 후 캘린더가 즉시 갱신됩니다.

9. 요약 뷰(📊 요약)

  • 선택된 기간의 총 매출을 표시합니다.
  • 셀러 기준 / 상품 기준을 전환하여 집계 단위를 변경합니다.
  • 매출 내림차순 랭킹 및 점유율(%)을 표시합니다.

10. 엑셀 저장/불러오기

10.1 저장(엑셀 내보내기)

  • 상단 저장을 클릭하면 엑셀 파일이 다운로드됩니다.
  • 저장 컬럼은 셀러명 / 상품명 / 행사일 / 매출액(억원)입니다.

10.2 불러오기(엑셀/CSV 가져오기)

  • 상단 불러오기를 클릭하여 파일을 선택하십시오.
  • 불러오기 완료 후 데이터가 반영되며 페이지가 새로고침될 수 있습니다.
권장 운영 중에는 주기적으로 저장하여 파일 백업을 유지하십시오.

11. 캘린더 프린트(A4 인쇄)

  1. 상단 캘린더 프린트를 클릭하십시오.
  2. 용지 방향(세로/가로)을 선택하십시오.
  3. 프린트 실행을 클릭하여 A4 중앙 정렬로 출력하십시오.
인쇄는 캘린더 그리드를 이미지로 캡처하여 출력합니다. 이벤트가 많을 경우 표시가 축약될 수 있습니다.

12. 주의사항

  • 로컬 저장: 브라우저 데이터 삭제/시크릿 모드/다른 기기 사용 시 데이터가 유지되지 않을 수 있습니다.
  • 백업 권장: 업무 운영 시 저장을 주기적으로 수행하십시오.
  • 초기화 주의: 초기화는 모든 데이터를 영구 삭제합니다.
주의 초기화는 되돌릴 수 없습니다. 필요 시 초기화 전에 반드시 엑셀로 저장하십시오.
